# Bounty Verification Bot

Auto-verifies bounty claims on Scottcjn/rustchain-bounties

## Features
- Star/follow verification via GitHub API
- Wallet existence check via RustChain node
- Article/URL verification
- Dev.to word count check
- Duplicate claim detection

## Payout Tiers (75 RTC max)
- Star/follow verification: 30 RTC
- Wallet check: +10 RTC  
- Article/URL verification: +10 RTC
- Dev.to quality check: +10 RTC
- Duplicate detection: +15 RTC
